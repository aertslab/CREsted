## Installation

If you have conda installed you can simply run:

```bash
make requirements  # Linux
```

This will create a conda environment named *DeepLocus*, or update the requirements in this environment if it already exists.

If not, create an environment manually following the package versions in *environment.yml*.


## Helpers

Useful helper functions for Linux can be found using:

```bash
make help
```

For example, you can delete all compiled Python files by calling `make clean`.
